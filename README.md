# agentaisec.github.io

Source for the [AgentAISec](https://agentaisec.github.io) website.

## Preview locally

To simply preview the website locally (*privately*), run:

```bash
cd agentaisec.github.io/ && python3 -m http.server 8080
```

Then open [http://localhost:8080](http://localhost:8080) in your browser.
